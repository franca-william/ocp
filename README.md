#Anotações
<br/>
<p> Variáveis de interface são public static final </p>
<p> Default métodos de interface são public </p>
<p> Static métodos de interface são public </p>
<p> Métodos privados não podem ser herdados </p>
<p> Protected pode ser acessado por todos do pacote e todos que herdam </p>
<p> Default pode ser acessado apenas por todos do pacote </p>
<p> Variáveis locais só podem ter o modificador de acesso final </p>
